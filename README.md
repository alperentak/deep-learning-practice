# Deep Learning Practice

Bu proje, farklı veri setleri ve modeller üzerinde çalışarak derin öğrenme
alanında pratik yapmak amacıyla oluşturulmuştur.

## Kurulum ve Başlangıç

Proje ve paket yönetimi için [uv](https://github.com/astral-sh/uv) aracı
kullanılmıştır.

```bash

# depoyu klonlayın
$ git clone https://github.com/alperentak/deep-learning-practice.git

# proje dizinine geçin
$ cd deep-learning-practice/

# bağımlılıkları kurun
$ uv sync
```
